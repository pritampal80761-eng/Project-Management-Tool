# Project Management Tool

A simple Project Management Tool built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## Features

* User Registration
* User Login
* Create Tasks
* View Tasks
* Mark Tasks as Completed
* MongoDB Database Integration
* REST API using Express.js

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB

## Installation

1. Clone the repository

git clone YOUR_GITHUB_REPOSITORY_LINK

2. Open backend folder

cd backend

3. Install dependencies

npm install

4. Start server

npm start

5. Open frontend files using Live Server
